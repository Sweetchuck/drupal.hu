## Requirements

1. `php --version` >= 7.1
1. `node --version` >= 9


## Getting started

1. Run `git clone`
1. Run `composer install`
1. Run `$EDITOR docroot/sites/default/settings.local.php`
1. Run `bin/robo site:install`
1. Run `cd docroot && php -S 127.0.0.1:8888 .ht.router.php`
1. Open `http://localhost:8888`
